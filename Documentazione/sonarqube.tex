SonarQube è una piattaforma di analisi statica del codice che permette di valutare automaticamente la qualità
del software durante lo sviluppo. Analizza il codice sorgente per individuare bug, vulnerabilità di sicurezza,
code smell e problemi di manutenibilità, fornendo metriche e report dettagliati.
Sonarqube fornisce una dashboard dove possiamo monitorare la qualità del codice: sicurezza,
leggibilità, manutenibilità.
Il progetto DietiEstates2025 ha superato con successo i test di qualita di SonarQube
